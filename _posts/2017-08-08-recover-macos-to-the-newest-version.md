之前重装 macOS 总遇到一个问题：Recovery 模式恢复的系统不是最新的版本。

今天得到了答案/解决办法：**⌥-⌘-R** （需要本机已安装 macOS Sierra 10.12.4 及以上版本）

| 组合键     | 效果                                       |
| ------- | ---------------------------------------- |
| ⌘-R     | 重新安装此 Mac 安装过的最新 macOS，但不会升级到更高的版本。（现在就好） |
| ⌥-⌘-R   | 升级到与此 Mac 兼容的最新 macOS。（升级到最新）            |
| ⇧-⌥-⌘-R | 重新安装 Mac 随附的 macOS 或仍有提供的最接近版本。（回到更老）    |
